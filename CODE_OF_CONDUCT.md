# FNLLA Code of Conduct

FNLLA is maintained as a professional working environment by TechAyo LTD (techayo.co.uk).

Everyone participating in repository work, issue handling, delivery hardening, review or maintenance is expected to help keep the working environment respectful, focused and commercially responsible.

## Expected behavior

- communicate clearly and professionally
- keep review feedback specific, actionable and respectful
- protect client, business and security-sensitive information
- respect repository ownership, product boundaries and licensing constraints
- raise risk early rather than hiding uncertainty

## Unacceptable behavior

- harassment, intimidation or personal attacks
- discriminatory or abusive language
- deliberate sabotage of repository quality, release metadata or ownership signals
- publishing private, client, business or security-sensitive information without authorization
- misrepresenting FNLLA or FNLLA Runtime ownership, authorship or licensing

## Maintainer responsibilities

TechAyo LTD and repository maintainers may:

- remove or edit content that conflicts with this code of conduct
- pause or close discussions that are no longer constructive
- restrict repository participation when behavior creates delivery, legal or security risk

## Scope

This code of conduct applies to:

- repository discussions and collaboration
- issue and review workflows
- release, maintenance and support conversations
- other spaces where someone is representing FNLLA or TechAyo LTD in relation to this repository

## Reporting

If you need to report unacceptable behavior, use TechAyo LTD's contact route at `https://techayo.co.uk` and clearly label the message `FNLLA Code of Conduct`.
